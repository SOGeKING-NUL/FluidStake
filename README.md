FluidStake is a decentralized wallet that continuously "autostakes" a fixed percentage of a user's funds while always keeping a matching liquid reserve. The idea is to automatically earn staking rewards without locking the user's funds straight from the wallet itself. 
When a user initiates a withdrawal that exceeds the current liquid balance, the wallet will "unstake" just enough from the staked portion to cover the shortfall-and then rebalance so that (for example) 50% of the remaining funds remain staked, and 50% stay liquid. The extra gas cost incurred by unstaking is effectively offset by the interest earned on the staked funds.
